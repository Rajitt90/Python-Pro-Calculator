# 🐍 Python Pro Calculator
A sleek, modular desktop calculator built using Python and the Tkinter library.

## 🚀 Overview
This project demonstrates the transition from a basic script to a structured, modular application. It features a professional UI inspired by modern aesthetics and a robust "state-machine" backend.

## 🛠️ Key Features
- **Modular Architecture:** Fully separated Frontend (`main.py`) and Backend (`logic.py`).
- **Modern UI:** Styled with a custom color palette (Dark Slate, Tan, and Orange).
- **Advanced Logic:** - Handles multi-step arithmetic operations.
  - Includes a "Clear" (C) function for resetting states.
  - Built-in error handling for division by zero.

## 📂 Project Structure
- `main.py`: Handles the Tkinter window, button layouts, and user interactions.
- `logic.py`: The "brain" of the app; processes numbers and manages the mathematical state.

## 💻 How to Use
1. Clone this repository to your local machine.
2. Ensure you have Python installed.
3. Run the application by executing:
   `python main.py`
